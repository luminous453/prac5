# Практическая работа №5

### Инструкция
1. Клонируем проект с файлами 
    ```shell
    git clone https://github.com/luminous453/prac5
   ```
2.
    ```shell
    npm install
   ```
3. Запускаем сервер 
    ```shell
    node server.js
   ```

4. Открываем HTML страничку в браузере, регистрируемся, логинимся, получаем токен <3
